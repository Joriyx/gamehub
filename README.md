# Gamehub

An online store for a fictional company that sells video games. Has a functional cart, and mocked checkout functionality.

Live [website](https://joriyx.github.io/gamehub/)

## Features

- Products fetched from external API
- Home page with popular products
- Shop with a grid of products
- Individual product page
- Mocked checkout functionality

## Prerequisites

- Visual Studio Code
- Live Server (VS Code extension)

## Getting Started

### Running the project

Click the 'Go Live' button in VS Code

## Technologies

- JavaScript
- HTML
- CSS
